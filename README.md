## 🔧 Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/sheethal5shivakumar/React-Portfolio--Sheethu.git
   ```

2. Install dependencies:
   ```bash
   cd React-Portfolio--Sheethu
   npm install
   ```

3. Start development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
